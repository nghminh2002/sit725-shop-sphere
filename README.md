# EVAT-Mobile/App
- Company: Chameleon
- Project: EV Adoption Tools
- Team: Mobile/App

### Tech Stack
- NodeJS
- TypeScript
- Express.js
- MongoDB
- JWT

### Env Variables

Create a .env file in then root and add the following

```
PORT = 8080
MONGODB_URI = MongoDB Uri
JWT_SECRET = 'abc123'
```

## Install Dependencies

```
npm install

```

## Run App

```

# Run in dev mode

npm run server

```
