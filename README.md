### Tech Stack
- NodeJS
- Express.js
- MongoDB
- JWT

### Env Variables

Create a .env file in then root and add the following

```
PORT = 8080
MONGODB_URI = MongoDB Uri
JWT_SECRET = 'ShopSphere'
```

## Install Dependencies

```
npm install

```

## Run App

```

# Run in dev mode

npm run start

```
